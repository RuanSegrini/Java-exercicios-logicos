
# Matrizes

## Objetivo
Praticar o uso de matrizes (arrays bidimensionais) em Java para resolver
problemas que envolvem tabelas, grades e relações entre linhas e colunas.

## Conceitos praticados
- Matrizes (arrays bidimensionais)
- Percorrer linhas e colunas
- Cálculo de soma, média, maior e menor valor
- Operações por linha e por coluna

## Regras adotadas
- Java puro (console)
- Uso de laços aninhados
- Código organizado e comentado
