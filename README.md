# Java - Exercícios de Lógica

Repositório criado para praticar lógica de programação em Java,
do nível básico ao intermediário.

## Conteúdos
- Estrutura Sequencial
- Condicionais
- Laços de Repetição
- Vetores
- Matrizes
- Métodos
- POO Básica

## Metodologia
- Resolver 2 exercícios por dia
- Um commit por sessão de estudo
- Código simples e bem comentado
