
# Métodos (Funções)

## Objetivo
Praticar a criação e utilização de métodos para organizar o código,
evitar repetição e melhorar a legibilidade dos programas em Java.

## Conceitos praticados
- Criação de métodos
- Parâmetros
- Retorno de valores
- Métodos `static`
- Reutilização de código

## Regras adotadas
- Java puro (console)
- Métodos bem nomeados
- Separação clara entre lógica e entrada/saída
- Código comentado
