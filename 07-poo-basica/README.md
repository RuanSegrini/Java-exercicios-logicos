
# Programação Orientada a Objetos (Básico)

## Objetivo
Praticar os conceitos básicos de POO em Java.

## Conceitos praticados
- Classes e objetos
- Atributos
- Métodos
- Construtores simples
- Encapsulamento básico

## Regras adotadas
- Código organizado em classes
- Sem frameworks
- Foco em clareza
