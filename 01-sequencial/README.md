
# Estrutura Sequencial

## Objetivo
Praticar a lógica de programação em Java utilizando operações básicas,
entrada e saída de dados e cálculos simples.

## Conceitos praticados
- Variáveis
- Operadores aritméticos
- Entrada e saída de dados (Scanner)
- Cálculos matemáticos

## Regras adotadas
- Java puro (console)
- Um exercício por pasta
- Código simples e comentado
